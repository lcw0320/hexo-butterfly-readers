官方文档

1. 快捷键

命令 快捷键
黑体 ctrl + B
斜体 ctrl + i
预览 ctrl k + v
完成任务清单 alt + C
表格对齐 alt + shift + F

2. 目录

2.1. 快速创建目录索引

  1. ctrl + shift + P打开vscode命令搜索栏
  2. 搜索Create Table of Contents回车

2.2. 快速添加删除目录数字标题

  1. ctrl + shift + P打开vscode命令搜索栏
  2. 搜索section,会出现相应的命令，回车。

3. 自动补全

3.1. 超链接

ctrl + c 复制超链接，然后选择指定字体ctrl + v，就可以将指定字体粘贴为超链接。
